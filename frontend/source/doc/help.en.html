File menu

Note: AGRAMMON always saves all changes to an input parameter directly to the database. Therefore there is no need to save your modifications when you are done. On the other hand, this means that you are always changing the current dataset immediately. There is currently no undo functionality.

If you want to keep the current dataset and to try new parameter settings, you can first copy the current dataset using File->Clone dataset and continue with this copy.

AGRAMMON version 2.x
