Anmeldung

Zur Benutzung von Agrammon benötigen Sie ein Benutzerkonto, unter dem ihre Eingabedaten gespeichert werden. Sie können sich selbst ein Konto einrichten mit der Funktion Neues Konto erstellen (Knopf rechts unten auf der Anmelde-Maske).

Falls Sie Ihr Passwort zu einem bereits eingerichteten Konto vergessen haben, können Sie die Funktion Passwort neu setzen (auf der Anmeldemaske rechts, 2. Knopf von unten) verwenden.

Falls Sie Schwierigkeiten haben, ein Benutzerkonto einzurichten, senden Sie bitte eine Email an: support@agrammon.ch.
